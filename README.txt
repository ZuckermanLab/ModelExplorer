On-going research into systems-based modeling of molecular machines. 

For an overview of the general methods used, see the PLOS Comp. Bio. Paper by George et al:https://doi.org/10.1371/journal.pcbi.1007884 

This repository contains: 
a) /archive/manuscript: data and scripts needed to reproduce the PLOS Comp. Bio. manuscript  
b) /archive/old_scripts: older scripts which may be useful for future work


Note that the software in this repository have had limited testing, and are not recommended for general public use.
